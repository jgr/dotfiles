dotfiles
========

Home Sweet Home (directory).

How to Make all the Symlinks:
-----------------------------
```
$ ./link.sh
```

How to get Vim Working:
-----------------------

1. Setup [Vundle]:

     ```
     $ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
     ```

2. Launch Vim

3. Install all the Vundle Bundles!

  ```vim
  :BundleInstall
  ```
